# 项目协作规则

创建时间：2026-05-23 13:03:45 北京时间

## 稳定命名和内容

- 所有设计稳定命名和内容均使用中文。
- 命名格式采用“时间 + 大概内容”。
- 涉及专有名词时，可以保留英文描述，例如 API、CLI、FastAPI、LightGBM。
- Python 代码中的包名、文件名、类名、函数名、变量名和测试名使用英文。
- Python 代码中可以适当使用中文注释、中文展示文案和中文测试样例数据。

## Git 提交

- Git 提交记录使用中文。

## Python 运行环境

- 本地 Python 解释器路径固定为：`C:\ProgramData\anaconda3\python.exe`。

## 时间规则

- 项目中所有涉及时间的展示、记录、配置和业务判断，均使用北京时间。
