# 规则推荐预览实现计划
创建时间：2026-05-23 17:45:00 北京时间

## 目标
新增第一版“规则基线”推荐预览，把已经生成的赔率特征转成亚盘和大小球的观望/倾向、信心等级和手数展示。

## 定位
本阶段推荐只作为开发期预览，不作为正式模型结论。它的作用是：

- 验证赔率特征是否能稳定驱动推荐输出。
- 给后续历史样本学习提供推荐结果的展示形态。
- 在没有历史回测前保持保守：分歧大、样本少、信号弱时观望。

## 规则
- bookmaker 数小于 6：观望，风险标签 `low_bookmaker_count`。
- 亚盘分歧大于 1.00：观望，风险标签 `handicap_disagreement_high`。
- 大小球分歧大于 0.75：观望，风险标签 `total_disagreement_high`。
- 亚盘方向：
  - 主队平均赔率低于客队平均赔率至少 0.12：倾向主队。
  - 客队平均赔率低于主队平均赔率至少 0.12：倾向客队。
  - 差距不足：观望。
- 大小球方向：
  - 大球平均赔率低于小球平均赔率至少 0.12：倾向大球。
  - 小球平均赔率低于大球平均赔率至少 0.12：倾向小球。
  - 差距不足：观望。
- 第一版 edge 使用赔率差距的保守代理值，并复用现有 `build_recommendation_from_signal` 映射信心等级和手数。
