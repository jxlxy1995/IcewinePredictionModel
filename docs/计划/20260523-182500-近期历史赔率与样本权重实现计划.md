# 近期历史赔率与样本权重实现计划
创建时间：2026-05-23 18:25:00 北京时间

## 目标
补齐最近已完赛比赛的赔率快照，并让训练样本带上时间衰减字段，支持后续“近高远低”的训练策略。

## 范围
本阶段完成：

- 新增 `sync historical-odds --days N`。
- 从数据库选择最近 `N` 天内已完赛、带 `source_match_id` 的比赛。
- 按开赛时间由近到远请求赔率。
- 能拿到赔率就保存，拿不到赔率则跳过。
- 遇到 API 限制错误时保留已成功结果并停止。
- 训练样本新增：
  - `sample_age_days`
  - `time_decay_weight`
  - `has_odds_snapshot`
- `samples preview` 展示这些字段。

## 时间衰减规则
- 0 到 180 天：`1.00`
- 181 到 365 天：`0.80`
- 366 到 730 天：`0.55`
- 731 到 1095 天：`0.35`
- 1096 天及以上：`0.15`

## 边界
本阶段不做：

- 多数据源历史赔率购买或接入。
- 训练样本落库。
- 真实机器学习训练。
- 历史赔率缺失的复杂重试队列。

## 验证
- 单测覆盖 fixture 选择窗口和排序。
- 单测覆盖空赔率跳过与 API 错误部分落库。
- 单测覆盖样本年龄、权重和有无赔率标记。
- 真实运行小窗口 `sync historical-odds --days 3`。
- 全量测试通过后中文提交。
