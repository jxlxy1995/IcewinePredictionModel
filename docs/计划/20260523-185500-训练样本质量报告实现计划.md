# 训练样本质量报告实现计划
创建时间：2026-05-23 18:55:00 北京时间

## 目标
新增训练样本质量报告，帮助在正式训练前判断当前数据池是否足够、是否偏科、赔率样本占比是否可用。

## 范围
本阶段完成：

- 新增样本报告服务。
- 统计总完赛样本数。
- 统计有赔率样本数和占比。
- 按联赛统计样本数。
- 按赛季统计样本数。
- 按时间权重统计样本数。
- 新增 CLI：`samples report`。

## 边界
本阶段不做：

- 图表。
- Web 页面。
- 训练集自动切分。
- 模型训练。

## 验证
- 单测构造多联赛、多赛季、有无赔率样本。
- CLI 单测覆盖 `samples report` 命令和格式化输出。
- 真实本地报告冒烟。
- 全量测试通过后中文提交。
