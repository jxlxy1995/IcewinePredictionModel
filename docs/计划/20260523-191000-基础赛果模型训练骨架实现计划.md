# 基础赛果模型训练骨架实现计划
创建时间：2026-05-23 19:10:00 北京时间

## 目标
新增第一版基础赛果模型训练与评估闭环，为后续逻辑回归、随机森林、梯度提升树等模型提供稳定对照组。

## 范围
本阶段完成：

- 从训练样本中按时间切分训练集和验证集。
- 基于训练集计算主队平均进球和客队平均进球。
- 使用泊松分布枚举比分，输出主胜/平/客胜概率。
- 在验证集上计算：
  - 样本数
  - 准确率
  - 平均 log loss
- 新增 CLI：`models train-baseline --limit N`。

## 边界
本阶段不做：

- sklearn 依赖。
- 模型持久化。
- 球队强弱特征。
- 赔率特征训练。
- 亚盘/大小球监督模型。

## 验证
- 单测覆盖平均进球训练。
- 单测覆盖泊松概率归一和结果方向。
- 单测覆盖验证集评估。
- CLI 单测覆盖输出格式。
- 真实本地训练冒烟。
