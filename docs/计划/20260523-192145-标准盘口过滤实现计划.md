# 标准盘口过滤实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 只允许 `X`、`X.25`、`X.5`、`X.75` 形式的亚盘和大小球盘口线进入聚合与推荐，异常盘口线先舍弃。

**Architecture:** 在 API 赔率映射层过滤非标准盘口，避免新数据入库；在特征聚合层再次过滤已存在的异常盘口，避免历史脏数据影响推荐。赔率字段本身不做标准步进过滤，只跟随对应盘口线是否有效参与市场聚合。

**Tech Stack:** Python 3.12、Decimal、pytest、SQLAlchemy。
