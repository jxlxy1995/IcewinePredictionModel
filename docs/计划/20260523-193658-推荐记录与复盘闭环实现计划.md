# 推荐记录与复盘闭环实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 建立推荐记录、赛后结算和 ROI 报表闭环，让模型能持续复盘自身推荐质量。

**Architecture:** 新增 `RecommendationRecord` 数据表保存正式推荐快照；新增 `record_service.py` 负责记录、查询、结算和报表。CLI 新增 `records` 命令组，并在 `recommendations record` 中复用现有模型推荐链路，只保存 `should_bet=True` 的推荐。

**Tech Stack:** Python 3.12、SQLAlchemy、Typer、Decimal、pytest、Asia/Shanghai 时间。
