# 历史表现接入推荐解释实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在推荐预览文本中展示类似历史样本数和 ROI，且不改变推荐决策。

**Architecture:** 扩展 `Recommendation` 增加只展示用字段，新建推荐历史表现增强函数。增强函数在推荐生成之后调用 `historical_performance_service`，返回带历史表现展示信息的新推荐列表。

**Tech Stack:** Python 3.12、dataclasses、Decimal、SQLAlchemy、pytest、Typer。
