### Task 5: OddsPapi 历史赔率接入实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 小样本、低成本接入 OddsPapi 历史亚盘和大小球赔率。

**Architecture:** 新增 OddsPapi client/provider/mapper，先将 OddsPapi fixture 与本地 match 映射，再抓取并过滤历史赔率。所有请求通过统一预算控制，历史赔率写入独立表，避免污染 API-Football 当前赔率快照。

**Tech Stack:** Python 3.12、requests、SQLAlchemy、Typer、pytest、SQLite。
