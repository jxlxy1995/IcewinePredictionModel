# OddsPapi批量回填控制器实现计划

## 目标

新增一个可通过 CLI 调用的 OddsPapi 批量回填控制器，支持按联赛队列、小批次循环、低并发 worker、连续空数据停止、无进展停止和汇总输出。

## 文件

- 新增 `src/icewine_prediction/oddspapi_batch_backfill_service.py`：批量控制器主体。
- 修改 `src/icewine_prediction/oddspapi_sync_runner.py`：提供结构化 `run_oddspapi_sync_result` 包装。
- 修改 `src/icewine_prediction/cli.py`：新增 `oddspapi-batch-backfill` 命令。
- 新增 `tests/test_oddspapi_batch_backfill_service.py`：控制器行为测试。
- 修改 `tests/test_oddspapi_sync_cli.py`：CLI 参数测试。

## 步骤

1. 写控制器单元测试，覆盖联赛排序、并发模式配置、无进展停止、连续空数据停止和汇总文本。
2. 运行测试，确认因为模块或函数不存在而失败。
3. 实现 `oddspapi_batch_backfill_service.py` 的数据类、模式解析、控制器循环和格式化输出。
4. 增加结构化同步入口 `run_oddspapi_sync_result`，复用既有同步逻辑。
5. 接入 CLI 命令 `odds-source oddspapi-batch-backfill`。
6. 运行相关测试，确认通过。

## 默认参数

- `--mode balanced`
- `--chunk-size 20`
- `--from-date 2026-01-15`
- `--max-rounds-per-league 20`
- `--stop-after-empty-matches 8`
- `--max-retries-per-league 2`
- `--request-budget-per-league 800`
- `--max-snapshots-per-match 120`
