# Web 控制台一期接力说明

## 工作边界

本对话只负责 Web 控制台开发，不负责 OddsPapi worker 巡检、回填收尾和未匹配别名矫正。worker 相关运行状态由另一条对话处理。

## 今晚目标

先让项目具备可运行的 Web 控制台骨架：

- 后端提供 FastAPI API 契约。
- 前端提供 React/Vite 控制台页面。
- 前端先使用 mock 数据。
- 后续公司环境可以继续做页面细化，不需要 `.env`、真实 SQLite 或 API key。

## 已新增后端入口

文件：

- `src/icewine_prediction/web_api.py`
- `src/icewine_web.py`
- `tests/test_web_console_api.py`

启动命令：

```powershell
$env:PYTHONPATH='src'
C:\ProgramData\anaconda3\python.exe -m uvicorn icewine_web:app --reload --host 127.0.0.1 --port 8000
```

已定义接口：

```text
GET /api/health
GET /api/dashboard/summary
GET /api/leagues/coverage
GET /api/workers
GET /api/unmatched
GET /api/matches/{match_id}/odds-trends
```

后端接口说明：

- `dashboard/summary`：返回比赛数、完赛数、历史赔率覆盖场次、赔率快照数、未匹配数。
- `leagues/coverage`：按联赛和赛季返回完赛场次、赔率覆盖场次、覆盖率、未匹配数。
- `workers`：读取 `logs/odds/workers/*.json`，如不存在则兼容读取 `logs/odds/oddspapi-worker-current.json`。
- `unmatched`：返回 `odds_source_matches.historical_odds_status = unmatched` 的比赛。
- `odds-trends`：返回指定比赛的亚盘和大小球历史快照。
