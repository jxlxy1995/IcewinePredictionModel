**Goal:** 新增负二项总进球模型，支持训练总进球均值/离散度，并计算大小球覆盖概率。

**Architecture:** 新建 `negative_binomial_model_service.py`，使用 `scipy.stats.nbinom` 计算总进球分布。训练函数只依赖 `TrainingSample.total_goals` 和 `time_decay_weight`；CLI 提供训练命令和纯参数调试命令。

**Tech Stack:** Python 3.11+、Decimal、scipy、pytest、Typer。
