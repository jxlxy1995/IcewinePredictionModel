# Web 控制台推荐复盘 mock 增强实现计划

## 背景

模型训练概览页已经能展示模型侧训练状态，但推荐记录页仍只有基础列表。为了让 Web 控制台在无真实数据库环境下也能演示推荐复盘闭环，本轮增强推荐记录 mock 和前端汇总能力。

## 范围

- 新增推荐记录复盘汇总纯逻辑，计算推荐数、已复盘数、手数、盈亏、ROI 和命中率。
- 按盘口类型、信心等级、联赛生成分组表现。
- 扩充 mock 推荐记录，覆盖已复盘、待复盘、赢、输、赢半、输半等状态。
- 推荐记录页面增加概览指标卡和分组表。
- 保持现有推荐记录表，并补充赛果展示列。

## 不做

- 不读取本地 SQLite。
- 不改后端推荐记录 API。
- 不改推荐结算规则。
- 不触发真实模型训练或真实回测。

## 验证

- `cd web && npm run test`
- `cd web && npm run build`
