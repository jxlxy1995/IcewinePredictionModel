# Web 控制台模型训练概览 mock 实现计划

## 背景

中文名维护页和 mock fallback 已完成后，Web 控制台下一步接入模型侧可见能力。当前真实训练数据和后端训练接口尚未稳定，因此先用 mock 数据展示模型训练状态、样本覆盖和最近训练结果。

## 范围

- 新增模型训练概览数据结构，覆盖模型版本、状态、训练样本数、联赛数、验证指标和盘口适用范围。
- 新增模型训练页导航入口。
- 使用 mock 数据展示 Dixon-Coles 攻防强度、Skellam 净胜球和负二项总进球三类模型。
- 展示训练样本、赔率快照、模型版本、可出结果数量。
- 展示联赛训练覆盖率。
- 增加前端纯逻辑测试，覆盖模型状态、最近训练排序和概览指标。

## 不做

- 不触发真实训练。
- 不读取本地 SQLite。
- 不设计新的后端训练 API。
- 不修改 OddsPapi 回填器和历史赔率抽样规则。

## 验证

- `cd web && npm run test`
- `cd web && npm run build`
