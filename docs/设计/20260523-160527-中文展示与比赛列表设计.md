# 中文展示与比赛列表设计

创建时间：2026-05-23 16:05:27 北京时间

## 一、目标

新增面向人工查看的中文展示层，让 CLI 输出的联赛名、球队名和比赛列表更符合中文阅读习惯。

程序内部、模型分析、API 映射和数据库主字段继续使用英文，保证稳定维护和外部数据源兼容。中文只用于展示层、文案、注释和测试样例。

## 二、展示名映射

新增 `config/display_names.yaml`，保存联赛和球队的中文展示名。

映射规则：

- 联赛名按 API-Football 英文名匹配，例如 `Serie A -> 意甲`。
- 球队名按 API-Football 英文名匹配，例如 `Inter -> 国际米兰`。
- 找不到映射时回退原英文名，不阻塞同步或展示。
- 映射配置可以持续补充，不影响历史数据。

## 三、比赛列表 CLI

新增命令：

```powershell
icewine matches upcoming --hours 24
```

输出格式：

```text
意甲 2026-05-24 00:00 博洛尼亚 vs 国际米兰
西甲 2026-05-24 03:00 皇家马德里 vs 毕尔巴鄂竞技
```

比赛时间使用北京时间。

## 四、时间窗口

`matches upcoming` 使用精确时间窗口过滤，而不是只按日期展示。

默认展示未来 24 小时内的未开赛比赛。

过滤规则：

- 起点：当前北京时间。
- 终点：当前北京时间 + `--hours`。
- 只展示 `status = scheduled` 的比赛。
- 按开赛时间升序排列。
