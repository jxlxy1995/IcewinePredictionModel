# Skellam 净胜球模型设计

创建时间：2026-05-25 北京时间

## 一、目标

本阶段新增 Skellam 净胜球模型，用于从主客队期望进球直接计算净胜球分布和亚盘覆盖概率。

Skellam 模型不替代现有比分矩阵模型，而是作为亚盘概率层的独立候选算法。它可以和现有从比分概率矩阵枚举亚盘概率的结果互相对照。

## 二、输入与边界

输入只需要：

- 主队期望进球
- 客队期望进球
- 亚盘盘口线
- 推荐方向：主队或客队

本阶段不读取赔率快照，不使用 24 小时赔率走势，不训练盘口价值模型，也不改变默认推荐逻辑。

## 三、模型形式

如果：

```text
home_goals ~ Poisson(home_expected_goals)
away_goals ~ Poisson(away_expected_goals)
```

则：

```text
margin = home_goals - away_goals
margin ~ Skellam(home_expected_goals, away_expected_goals)
```

亚盘覆盖概率通过净胜球分布直接计算：

- 赢盘记为 `1`
- 走水记为 `0.5`
- 输盘记为 `0`
- 四分之一盘口拆成两个半盘口后取平均

## 四、模块设计

新增 `skellam_model_service.py`。

核心类型：

```python
SkellamMarginModel
SkellamHandicapProbability
```

核心能力：

- `margin_probability(margin)`
- `home_cover_probability(line)`
- `away_cover_probability(line)`
- `asian_handicap_probability(line)`

底层使用 `scipy.stats.skellam`，保持和现有 `scipy` 依赖一致。

## 五、CLI 设计

新增命令：

```powershell
icewine models skellam-handicap --home-eg 1.50 --away-eg 1.10 --line -0.25
```

输出：

- 主队期望进球
- 客队期望进球
- 盘口线
- 主队覆盖概率
- 客队覆盖概率

该命令用于工程验证和调试，不读取数据库。

## 六、明确不做

本阶段不做：

- 不训练 Skellam 参数。
- 不接入默认推荐。
- 不接入赔率走势。
- 不计算真实投注收益。
- 不替代现有比分矩阵概率服务。

后续如果需要，可以把 Skellam 输出作为推荐解释字段，或在回测中和比分矩阵法对比。
